### Код лотореи "шар" осеннего семестра 2017
{warning:title:Внимание!|icon=False}
Убедись в наличии Python 3.* и NumPy
{warning}
---------
Для установки NumPy:
```
$ pip install numpy
```
или
```
$ pip3 install numpy
```
----------
Для того, чтобы запустить файл, открой териминал, зайди в папку, со скачаным файлом и:
```
$ python3 luckyme.py
```
