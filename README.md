### Код лотореи "шар" осеннего семестра 2017
-----
> Для запуска требуется установленый Python 3.* и NumPy

-----
Для установки NumPy:
```
$ pip install numpy
```
или
```
$ pip3 install numpy
```
-----
Для того, чтобы запустить файл, открой териминал, зайди в папку, со скачаным файлом и:
```
$ python3 luckyme.py
```
